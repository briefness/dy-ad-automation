# 库布里克风格（Kubrick Style）

> 对称构图大师，以「单点透视」「前灯光」和「精确到帧的构图」闻名，画面充满仪式感和宿命感。

## 核心运镜

### 1. 单点透视推进（One-Point Perspective Dolly）
```
Kubrick one-point perspective: camera pushes straight down 
center of symmetrical corridor, everything aligns to vanishing 
point, 2001: A Space Odyssey style, cosmic dread, meticulous 
symmetry, fluorescent lighting
```
**适用片段**：展示（片段3）
**效果**：产品展示充满仪式感

### 2. 前灯光（Frontal Lighting / Key Light）
```
Kubrick frontal lighting: strong key light from front, deep 
shadows on sides of face, dramatic chiaroscuro, Barry Lyndon 
candlelight style, natural window light
```
**适用片段**：全片通用
**效果**：面部立体，情绪严肃

## 核心转场

### 1. 匹配剪辑（Match Cut）
```
Match cut from 2001: A Space Odyssey: cut from bone flying 
up to spaceship, matching shape and movement, Kubrick style, 
evolution theme, seamless transition
```
**适用**：片段3→4 概念升华
**效果**：从产品到情感的完美跳切

### 2. 时间跳跃（Time Jump）
```
Kubrick time jump: sudden cut from bright daylight to dark 
night, same location, same composition, only lighting changes, 
The Shining style, time distortion
```
**适用**：片段2→3 展示
**效果**：制造时间流逝感

## 色彩与光线
- **色调**：高饱和度红/蓝/绿，或暖黄烛光
- **光线**：前侧光为主，阴影锐利
- **参考影片**：《2001太空漫游》(2001: A Space Odyssey)、《巴里·林登》(Barry Lyndon)

## 抖音适配提示
- 保持画面绝对水平，避免倾斜
- 产品放置画面正中央
- 用强光制造戏剧性阴影
