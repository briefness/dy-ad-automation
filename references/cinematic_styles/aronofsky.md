# 阿伦诺夫斯基风格（Aronofsky Style）

> 心理惊悚大师，以「快速推轨」「分裂半透镜（Split Diopter）」和「密集剪辑」闻名，镜头充满焦虑和强迫感。

## 核心运镜

### 1. 快速推轨（Rapid Push-In）
```
Aronofsky rapid push-in: fast dolly toward subject's face, 
Requiem for a Dream style, claustrophobic tension, 
split diopter, foreground and background both sharp, 
frantic energy
```
**适用片段**：转折（片段2）
**效果**：紧迫感，从痛苦到聚焦

### 2. 分裂半透镜（Split Diopter）
```
Split diopter shot: foreground and background both in focus, 
Aronofsky style, Requiem for a Dream drug sequence, 
compressed space, paranoid atmosphere, shallow depth of field
```
**适用片段**：展示（片段3）
**效果**：产品细节+环境同时清晰

## 核心转场

### 1. 快速跳切（Rapid Jump Cut）
```
Aronofsky jump cut: rapid succession of similar frames, 
subject's expression intensifies, time compression, 
drug sequence style, disorienting
```
**适用**：片段1→2 痛点→转折
**效果**：从困惑到紧迫的情绪加速

### 2. 眼球转场（Eye Match Cut）
```
Eye match cut: cut from one character's eye to another's, 
or eye to object, Aronofsky style, connection theme, 
Pi/Requiem style, obsessive focus
```
**适用**：片段2→3 人物→产品
**效果**：视线引导，自然过渡

## 色彩与光线
- **色调**：高对比，冷绿/暖黄交替
- **光线**：硬光为主，阴影浓重
- **参考影片**：《梦之安魂曲》(Requiem for a Dream)、《圆周率》(Pi)

## 抖音适配提示
- 快速推轨适合前3秒钩子
- 分裂半透镜让产品细节和环境同时出镜
- 快速跳切适合抖音快节奏
