## 一、前期准备

### 1.1 素材清单

在开始前，确保以下素材已准备就绪：

```
项目文件夹/
├── 01_素材/
│   ├── character_ref.png          # 角色定妆照（PNG 无损格式）
│   ├── logo.png                   # 品牌 Logo（PNG 透明背景）
│   ├── bgm.mp3                    # 背景音乐（MP3 格式）
│   └── subtitles.txt              # 字幕文案（可选，剪映可自动识别）
├── 02_片段/
│   ├── clip_01_hook.mp4          # 0-5s 钩子
│   ├── clip_02_turn.mp4          # 5-10s 转折
│   ├── clip_03_show.mp4          # 10-18s 展示
│   ├── clip_04_result.mp4        # 18-25s 结果
│   └── clip_05_cta.mp4           # 25-30s CTA
├── 03_导出/
│   └── final_video.mp4
└── README.md
```

### 1.2 剪映项目设置

1. 打开剪映，点击「开始创作」
2. 比例选择 **9:16（竖屏）**
3. 分辨率选择 **1080×1920**
4. 帧率选择 **30fps**
5. 点击「创建项目」

## 二、可灵生成规范

### 2.1 生成前检查清单

在可灵 AI 中生成每个片段前，确认以下设置：

| 检查项 | 要求 | 说明 |
|--------|------|------|
| 分辨率 | 1080×1920 | 竖屏 9:16 |
| 帧率 | 24fps | 标准帧率 |
| 参考图 | 上传 character_ref.png | 保证人物一致性 |
| 种子（Seed） | 固定同一数值 | 全片风格一致 |
| 运动强度 | 0.5-0.7 | 避免人物变形 |
| 生成模式 | 标准模式 | 速度与质量平衡 |

### 2.2 片段生成顺序

**建议按以下顺序生成，便于后期衔接：**

1. **片段 1**（钩子）：固定→推进，0-5s
2. **片段 2**（转折）：手持跟拍，5-10s
3. **片段 3**（展示）：环绕，10-18s
4. **片段 4**（结果）：拉远，18-25s
5. **片段 5**（CTA）：固定，25-30s

### 2.3 单片段时长控制

| 片段 | 建议时长 | 可灵单次生成上限 | 备注 |
|------|---------|-----------------|------|
| 片段 1 | 5s | 10s | 钩子，节奏快 |
| 片段 2 | 5s | 10s | 转折，动作连贯 |
| 片段 3 | 8s | 10s | 展示，可最长 |
| 片段 4 | 7s | 10s | 结果，情绪高潮 |
| 片段 5 | 5s | 10s | CTA，简洁收尾 |

### 2.4 导出设置

每个片段生成后，导出时选择：

- 格式：MP4（H.264）
- 分辨率：1080×1920
- 帧率：30fps
- 码率：15-20 Mbps
- 命名规范：`clip_01_hook.mp4`、`clip_02_turn.mp4` 等

## 三、剪映拼接步骤

### 3.1 导入素材

1. 将 5 个视频片段导入剪映媒体库
2. 按顺序拖到时间轴：
   ```
   片段1 → 片段2 → 片段3 → 片段4 → 片段5
   ```
3. 检查每个片段的入点和出点，确保无黑边

### 3.2 转场处理

**转场时间轴：**

```
片段1 (0-5s)  ──[黑场 0.3s]──▶  片段2 (5-10s)
片段2 (5-10s) ──[白场 0.5s]──▶  片段3 (10-18s)
片段3 (10-18s) ──[叠化 0.3s]──▶  片段4 (18-25s)
片段4 (18-25s) ──[Match Cut 0.3s]──▶  片段5 (25-30s)
```

**具体操作：**

| 转场位置 | 转场类型 | 时长 | 剪映操作 |
|---------|---------|------|---------|
| 片段1→2 | 黑场 | 0.3s | 添加「闪黑」转场 |
| 片段2→3 | 白场/光线 | 0.5s | 添加「闪白」转场 |
| 片段3→4 | 叠化 | 0.3s | 添加「叠化」转场 |
| 片段4→5 | Match Cut | 0.3s | 检查动作衔接，可加「缩放」转场 |

**操作步骤：**
1. 点击两个片段之间的「+」号
2. 选择「转场」
3. 搜索对应转场效果（如「闪黑」「闪白」「叠化」）
4. 调整时长到指定值
5. 预览检查是否自然

### 3.3 节奏检查

- 全片总时长控制在 **28-32s**（留 2s 余量）
- 每个转场点检查是否有跳帧或画面跳动
- 如有问题，微调转场时长或重新生成该片段

## 四、调色统一

### 4.1 统一 LUT

1. 下载一个适合抖音的 LUT（推荐：「电影暖调」「清新明亮」「INS 风」）
2. 在剪映中点击「调节」→「LUT」
3. 上传 LUT 文件，强度设为 **30-50%**
4. 应用到所有片段

### 4.2 逐段调整

| 调整项 | 目标 | 剪映操作 |
|--------|------|---------|
| 亮度 | 统一 ±5% 内 | 调节 → 亮度 |
| 对比度 | 统一 ±5% 内 | 调节 → 对比度 |
| 饱和度 | 统一 ±10% 内 | 调节 → 饱和度 |
| 色温 | 统一色调 | 调节 → 色温 |
| 肤色 | 自然红润 | 美颜 → 肤色均衡 |

### 4.3 快速调色技巧

- **使用「色彩匹配」**：剪映自动匹配片段间色调
- **套用「滤镜」**：选择统一滤镜，强度 20-30%
- **调整「曲线」**：微调高光和阴影，保持画面通透

## 五、字幕与 BGM

### 5.1 字幕设置

| 项目 | 设置值 | 说明 |
|------|--------|------|
| 字号 | 60-80 | 抖音最佳阅读字号 |
| 颜色 | 白色 + 黑色描边 | 高对比度，任何背景都清晰 |
| 位置 | 画面下方 1/3 | 避开人物面部 |
| 动画 | 弹出/打字机 | 吸引注意力 |
| 停留时间 | 2-3s | 与台词同步 |

**操作步骤：**
1. 点击「文本」→「新建文本」
2. 输入字幕内容
3. 调整字号、颜色、描边
4. 添加动画效果
5. 拖动调整时长与画面同步

### 5.2 BGM 设置

| 项目 | 设置值 | 说明 |
|------|--------|------|
| BGM 音量 | 60% | 人声/音效 40% |
| 节奏 | 120-140 BPM | 抖音热门节奏 |
| 卡点 | 每 2-3s 一个节拍 | 与转场点对齐 |
| 淡入淡出 | 前 1s 淡入，后 1s 淡出 | 避免突兀开始/结束 |

**操作步骤：**
1. 点击「音频」→「音乐」
2. 选择抖音热门 BGM 或上传自制音乐
3. 调整音量至 60%
4. 拖动音频到时间轴，对齐转场点
5. 添加淡入淡出效果

### 5.3 音效添加（可选）

| 音效 | 使用位置 | 效果 |
|------|---------|------|
| 鼓点 | 每个转场点 | 强调节奏 |
| whoosh | 快切转场 | 掩盖切换 |
| 叮/提示音 | CTA 出现 | 吸引注意 |
| 环境音 | 全片背景 | 增加真实感 |

## 六、导出设置

### 6.1 导出参数

| 参数 | 设置值 | 说明 |
|------|--------|------|
| 分辨率 | 1080×1920 | 竖屏 9:16 |
| 帧率 | 30fps | 标准帧率 |
| 码率 | 15-20 Mbps | 高清画质 |
| 格式 | MP4（H.264） | 抖音兼容格式 |
| 编码 | H.264 | 通用编码 |

### 6.2 导出步骤

1. 点击右上角「导出」
2. 选择「4K 导出」（实际输出 1080p）
3. 确认分辨率为 1080×1920
4. 帧率选择 30fps
5. 码率选择「高」或自定义 15-20 Mbps
6. 点击「导出」
7. 等待导出完成（约 2-5 分钟）

## 七、常见问题

### Q1：片段拼接处画面跳动怎么办？

**A：**
1. 检查前后片段的光线是否一致
2. 加 0.5s 黑场/白场转场掩盖
3. 如仍明显，重新生成该片段

### Q2：人物面部在不同片段中不一致？

**A：**
1. 确认所有片段都使用了同一张 `character_ref.png`
2. 确认 seed 值固定
3. 如问题严重，使用 Reface 或 DeepFaceLab 后期统一

### Q3：转场处有跳帧？

**A：**
1. 检查转场时长是否过短（建议 ≥0.3s）
2. 增加转场特效（如光效、模糊）掩盖
3. 重新设计转场动作，使前后镜头动作连续

### Q4：字幕被产品/人物遮挡？

**A：**
1. 调整字幕位置到画面下方
2. 或调整字幕大小，避开遮挡区域
3. 使用「字幕背景」功能添加半透明背景

### Q5：导出后画质压缩严重？

**A：**
1. 导出时选择「高码率」
2. 确保源素材分辨率 ≥ 1080×1920
3. 避免多次导出压缩（只导出一次最终版本）
